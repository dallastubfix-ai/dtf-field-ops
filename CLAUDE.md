# DTF Field Ops — Claude Code Context

## What This Is
Mobile-first PWA for Dallas Tub Fix field operations.
Repo: dtf-field-ops | Deployed: Vercel | DB: Supabase (DTF Gmail account)

## Stack
React + Vite | Tailwind CSS | React Router v6 | Supabase JS v2
Dexie.js v4 (offline IndexedDB) | vite-plugin-pwa | lucide-react | date-fns

## Supabase Tables
customers, jobs, job_number_seq, appointments, images, videos,
invoices, warranties, push_subscriptions

## Key Architecture Rules
- Offline-first: all writes go to Dexie first, then Supabase
- If offline: queue operation in sync_queue Dexie table, process when back online
- Auth: Google OAuth via Supabase, biometric via WebAuthn for subsequent logins
- Job numbers: auto-generated by generate_job_number() Postgres function (DTF-26006+)
- Images: stored in Supabase Storage bucket "job-images" (private)
- Videos: stored in Google Drive via Drive API
- PDF output: window.print() on InvoiceBuilder and WarrantyBuilder pages
